# Phase 1: Daemon + Postgres Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (inline) to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Postgres becomes source of truth (write-through cache preserving v0.1 retrieval semantics), served by a single MCP-over-HTTP daemon with a stdio shim; legacy `.pt` banks migrate automatically; band weights get atomic, hardened persistence.

**Architecture:** Spec §3–§4 of `2026-06-10-v0.2-daemon-postgres-design.md`. New `storage/` package behind a narrow protocol; CMS/CortexStore emit write-through calls at mutation points; the daemon is the only writer.

**Tech Stack:** psycopg3 (binary), Postgres 16 + pgvector + AGE in Docker (`ops/`), `mcp` SDK streamable-HTTP + uvicorn, stdio proxy shim.

**Branch:** `v0.2-phase1`. **Test command:** same system Python as Phase 0.
**Test DB:** `PSEUDOLIFE_TEST_DATABASE_URL=postgresql://pseudolife:pseudolife@127.0.0.1:5433/pseudolife_memory_test` (fixture creates/drops the DB; storage tests `pytest.skip` when unset AND the local container is absent).

### Task 3: PostgresStorage backend

**Files:** create `pseudolife_memory/storage/postgres.py`; extend `tests/test_pg_storage.py`.

Protocol (sync, single-connection, daemon-internal — the service lock serializes):

```python
class PostgresStorage:
    def __init__(self, dsn: str): ...           # connects, ensure_schema
    # entries
    def insert_entry(self, e: dict) -> int      # returns id
    def update_entry(self, id: int, **fields)   # band / superseded_* / access_count / tags / slots
    def delete_entry_ids(self, ids: list[int])
    def load_entries(self) -> list[dict]        # all, embedding as list[float]
    # episodes
    def upsert_episode(self, ep: dict); def load_episodes(self) -> list[dict]
    # cortex facts (full row per spec; status drives current/contender/superseded)
    def upsert_fact(self, f: dict) -> int; def load_facts(self) -> list[dict]
    def delete_fact_ids(self, ids: list[int])
    # meta
    def meta_set(self, key: str, value); def meta_get(self, key: str, default=None)
    def close(self)
```

- [ ] TDD CRUD round-trips for each group (embedding fidelity to 1e-6; JSONB tags/slots; fact status transitions).
- [ ] Commit.

### Task 4: Atomic weights persistence (independent of PG)

- [ ] `atomic_torch_save(obj, path)`: `torch.save` → `path.tmp` then rotate `path`→`path.bak`, `os.replace(tmp, path)`.
